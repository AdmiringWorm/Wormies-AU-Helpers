Wormies AU Helpers

What is it?

Wormies AU Helpers is a collection of helper scripts to make maintaining packages using AU even easier.

Where to get it?

No official release does yet exist, but when it does we will update this page with the location.

Deveopment versions can be installed through chocolatey with the command choco install wormies-au-helpers --source https://www.myget.org/F/wormie-nugets --pre, and can be viewed here. These versions can also be installed using the latest AppVeyor Artifacts.
